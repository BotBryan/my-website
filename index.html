Requirements

  • Wii or Wii U modded with CleanRip and the Homebrew Channel
  • GameCube disc
  • Windows Computer
  • Thumb Drive
  • Dolphin Emulator

Process

  1. Follow the Homebrew Setup Guide.
  2. Install CleanRip using the following guide.
  3. Rip the .iso off of the disc to a thumbdrive formatted in FAT through CleanRip on the Homebrew Channel.
  4. .iso files are compatible with most emulators, including dolphin.
  5. If you wish to convert the .iso into .rvz or .gcz, you can change it in the Dolphin file explorer.
    • Put the game file in Dolphins \game\ folder.
    • Right-click the .iso in the Dolphin window and select "Convert File..."
    • Select a format and hit "Convert..."
